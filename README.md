# cs-tutorial-platform

Angular-based tutorial platform for Computer Science topics (OOP, DSA, Spring Boot, System Design) with interactive components and visualizations.

## Setup

Run `npm install` to install dependencies.

## Development

Run `npm start` for a dev server. Navigate to `http://localhost:4200/`. The application reloads when you change source files.

## Build

Run `npm run build`. Client build output is in `dist/client`.

## Test

Run `npm run test` to run unit tests with [Vitest](https://vitest.dev).
